About this demo
This frontend shows how to connect a drone image analysis pipeline with an actionable recommendation UI. Replace the simulated detection with a backend ML API (send image, get plant & pest severity). The doses below are a simple formula example — do soil tests before acting.
Useful next steps: integrate real inference (TensorFlow / PyTorch), incorporate soil data, integrate weather & local regulations.
